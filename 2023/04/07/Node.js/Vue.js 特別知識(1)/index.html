Vue.js 特別知識(1) - 強制渲染畫面 this.$forceUpdate()

Vue.js 特別知識(1) - 強制渲染畫面 $forceUpdate()

問題描述：

使用 Vue 框架開發時，在某個 function 片段中的 this.driving_track 加入新的值

但是在畫面上卻無任何變化（沒有連動）
地址欄位正常要有資料

解決方式：

這時只需要加入 this.$forceUpdate() 強制更新即可（因為數據層太多，需要手動更新）
